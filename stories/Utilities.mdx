# Utilities

This is a group of utility classes allowing easy application of consistent design patterns across elements without the need for custom CSS for each instance.

## Spacing

The following demonstrates the various spacing values available in our design system.

<table>
  <thead>
    <tr>
      <th>Key</th>
      <th>Value</th>
      <th>Example</th>
    </tr>
  </thead>
  <tbody>
    {Object.entries({
      0: '0',
      1: '0.25rem',
      2: '0.5rem',
      3: '1rem',
      4: '1.5rem',
      5: '2rem',
      6: '4rem'
    }).map(([key, value]) => (
      <tr key={key}>
        <td>{key}</td>
        <td>{value}</td>
        <td>
          <div style={{ backgroundColor: '#e0e0e0', height: '1rem', width: value }}></div>
        </td>
      </tr>
    ))}
  </tbody>
</table>
