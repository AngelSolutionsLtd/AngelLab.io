# Utilities

This is a group of utility classes allowing easy application of consistent design patterns across elements without the need for custom CSS for each instance.

## Spacing Size

The following demonstrates the various spacing values available in our design system.

## Spacing Classes

Spacing classes include margin and padding.

Size values include: 1 - 6 from the above table and breakpoint values include sm, md, lg, and xl.

```html
.p-{breakpoint}-{size} // all sides
.m-{breakpoint}-{size} // all sides
.m-{breakpoint}-auto // auto

.pt-{breakpoint}-{size} // padding top
.mt-{breakpoint}-{size} // margin top
.mt-{breakpoint}-auto // margin top auto
.nmt-{breakpoint}-{size} // negative margin top

.pr-{breakpoint}-{size} // padding right
.mr-{breakpoint}-{size} // margin right
.mr-{breakpoint}-auto // margin right auto
.nmr-{breakpoint}-{size} // negative margin right

.pb-{breakpoint}-{size} // padding bottom
.mb-{breakpoint}-{size} // margin bottom
.mb-{breakpoint}-auto // margin bottom auto
.nmb-{breakpoint}-{size} // negative margin bottom

.pl-{breakpoint}-{size} // padding left
.ml-{breakpoint}-{size} // margin left
.ml-{breakpoint}-auto // margin left auto
.nml-{breakpoint}-{size} // negative margin left
```

Alignment classes can be used on their own or they can used with breakpoints

    </div>

## Borders

Each class applies a border to one side of an element, with varying thicknesses and responsive breakpoints.

<ul>
  <li>thickness - 1 - 9</li>
  <li>breakpoint - sm, md, lg, xl</li>
</ul>

```html
.border-{thickness}-{breakpoint}-left
.border-{thickness}-{breakpoint}-right
.border-{thickness}-{breakpoint}-top
.border-{thickness}-{breakpoint}-bottom
```

no-border class removes border from one side

```html
.no-border-{breakpoint}-left
.no-border-{breakpoint}-right
.no-border-{breakpoint}-top
.no-border-{breakpoint}-bottom
```

## Display

Display values including breakpoint values, Where value is one of:

<ul>
  <li>none</li>
  <li>inline</li>
  <li>inline-block</li>
  <li> block</li>
  <li>table</li>
  <li>table-cell</li>
  <li>table-row</li>
  <li>flex</li>
  <li>inline-flex</li>
</ul>
   

```html
.d-{value} for xs
.d-{breakpoint}-{value} for sm, md, lg, and xl.
```

## Sizing

width or height values include


<ul>

  <li>50%</li>

  <li>100%</li>
